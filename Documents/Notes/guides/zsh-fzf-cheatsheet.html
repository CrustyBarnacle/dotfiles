Guided exercises — try these in order
01
Press Ctrl+R and type "pacman"
Find a previous pacman command. Press Enter to run it, or Ctrl+C to cancel. Notice the preview of matching commands.
02
Press Ctrl+T from your home directory
Start typing "sway" — watch it filter files across your whole home dir. Press Esc to cancel.
03
Press Alt+C and type "waybar"
fzf finds all subdirectories matching "waybar" and cd's into the one you pick.
04
Type: cat ~/.config/sway/config
Notice bat's syntax highlighting instead of plain cat output.
05
Type: h sway (then try h rmpc)
Your h() function pipes --help output through bat for readable formatting.
06
Run any command, then type: echo !$
!$ expands to the last argument of your previous command. Useful for chaining.
07
Type a long command, then use Ctrl+A, Ctrl+E, Ctrl+W
Practice jumping to start/end of line and deleting words backwards. These become muscle memory fast.
08
Start typing "yadm" — wait for the grey suggestion, press →
Autosuggestions show the most recent matching command in grey. Arrow right accepts it.
Pro tip: The single most useful habit to build is Ctrl+R for history search. You'll stop retyping long commands immediately. After that, Ctrl+W (delete word) saves more keystrokes than almost anything else.
