neovim · lazyvim — Section-9

Space is your leader key. Bottom-left shows current mode. Escape always returns to Normal.

Modes — the most important concept
EscNormal Home base. Navigate, run commands.
iInsert Type text. Esc to leave.
vVisual Select text. Esc to leave.
VVisual Line Select whole lines.
:Command Run ex commands. Enter to run, Esc to cancel.

Tier 3 — Power moves (grow into these)
Text objects — the real power
ci"change inside quotes
ca"change around quotes (incl. quotes)
ci(change inside parentheses
citchange inside HTML tag
ciwchange inside word
di{delete inside braces
Pattern: verb + i/a + object. Works with d, c, y, v too.
Marks & jumps

Macros

Guided exercises — open any config file and try these
Exercises in order
01
nvim ~/.config/sway/config
Open a file. Move around with h/j/k/l. Press Esc if anything goes wrong. Type :q to quit.
02
Press i, type something, press Esc, then u to undo
The insert → normal → undo cycle is the foundation of everything.
03
Navigate to a line, press dd, then p to paste it elsewhere
dd cuts the line, p pastes it. This is how you move lines around.
04
Press / then type a word to search. Press n to jump to next match.
Search is how you navigate large files. Much faster than scrolling.
05
Press Space Space — type a filename fragment
This is LazyVim's fuzzy file finder. The most-used Space command.
06
Open a .lua or .py file. Move cursor onto a function name. Press K.
K shows hover documentation from the LSP. Press Esc to close.
07
Look at the relative numbers on the left. Type 5j to jump down 5 lines.
Relative numbers tell you exactly how far to jump. This becomes fast.
08
Position cursor inside quotes or parens. Type ci" or ci(
Change inside — deletes the contents and puts you in insert mode. One of the most useful text objects.
09
Press Space g g to open lazygit (if in a git repo)
Full git UI inside neovim. Press q to quit lazygit.
10
Press Space — just Space, then wait
A menu appears showing all available commands. This is your discovery tool — browse it whenever you're curious what's available.
The single best habit to build first: always return to Normal mode with Esc when you're not actively typing. In Normal mode, every key is a command. In Insert mode, every key is text. The switch between them is the core skill.
